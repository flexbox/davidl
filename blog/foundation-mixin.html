Liste des mixin Foundation 5

Certaines choses en CSS sont un peu fastidieuses à écrire

27/02/2015

Cette liste présente les composants de Fondation 5 avec toutes leurs mixins. Pour plus d'informations, cliquez sur chaque lien pour visiter la documentation.

  @include accordion-container;
  @include accordion-navigation;
  @include accordion-content;

Alerts

  @include alert;

Block grid

  @include block-grid;

  @include crumb-container;
  @include crumbs;

Buttons

  @include button;

Buttons, dropdown

  @include button;
  @include dropdown-button;
  @include radius;
  @include single-transition;
  @include inset-shadow;

Grid

  @include grid-row;

Labels

Pagination

  @include pagination;

Panels

  @include switch-style;

Thumbnails
