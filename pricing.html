Grille de tarification

Trouver une solution qui vous correspond

Pourquoi ?

Le but de cette grille de tarification est de proposer différentes gammes de budget pour mes services de freelance. Ces évaluations ne sont pas des bugdets définitifs, mais peuvent êtres considérés comme point de départ pour vous aider à choisir la meilleure solution possible en fonction de votre budget.

Comment lire cette grille ?

Les grilles ont différentes options disponibles comme le code front-end et le responsive design.

Encore des petites choses

  • Dans tous les cas, considérez que la cession des droits est comprise dans les tarifs.
  • Certains type de travaux ne sont pas inclus dans les budgets : stratégie du contenu, recherches, et autres type de travail préparatoire. Je considère que cela est déjà effectué en partie par mes clients. Si ce n'est pas le cas il est possible de réaliser une évaluation séparée pour ça.
  • Chaque catégorie est illustrée par un exemple réel de réalisation.

Conception de landing page

Proposer une page web simple (Vente d’un produit, site vitrine, sensibilisation ...)

ChtiSARL travaille sur un nouveau produit, ils ont embauché Jacques pour mettre en place une page web d’attente responsive. Après un commun accord sur les maquettes préléminaires, Jacques travaille sur une atmosphère avec un logiciel comme Photoshop ou Illustrator. Ensuite, après avoir effectué 3 cycle de révision avec le client il peut commencer le codage HTML et CSS.

  • Maquette préliminaire
  • Modèle de page d’accueil : 3 révisions
  • Livrables : Fichiers sources statiques (HTML et CSS)
  • 10 - 15 heures de travail

2000 - 3000 €

Design de site web responsive

Design complet de site internet (homepage, pages de contenu, etc.).

ChtiSARL souhaite mettre à jour leur présence en ligne. Ils ont embauché Alice. Elle crée 3 maquettes au format papier pour la page d’accueil, les pages produits et le blog. Ensuite elle conçoit un prototype haute fidélité de la page d’acceuil directement dans un navigateur, qui passera par une série de 3 validations.
Dès que la homepage est terminée, elle code les deux autres modèles, avant de livrer le résultat comme un ensemble de fichiers HTML et CSS.

  • 3 maquettes
  • Prototype principal : 3 révisions
  • 1 page de contenu (Blog, FAQ, ...)
  • 1 page bonus (Contact, services, ...)
  • Livrables : Des fichiers statiques HTML, CSS et JavaScript
  • 30 - 50 heures de travail

5500 - 7000 €
